**第 0002 题:**  冒泡排序（Bubble sort）  稳定
	第一趟：从a[0]开始，比较a[1]，如果a[0]大，则a[0],a[1] =a[1],a[0],继续比较a[[1]与a[2],直到a[max],排序后a[max]为最大值
	第二趟：从a[0]开始，直到a[max-1]
	第n趟: 从a[0]开始,直到a[max-n]
	最好情况:顺序 时间复杂度  T = O(n)	
	最坏情况:逆序 时间复杂度  T = O(n**2)
	
**第 0003 题:**  快速排序 (Fast sort)
		

**第 0004 题:**  希尔排序（Shell sort）
	第一趟:	5间隔:对每个数进行5间隔排序,即a[0],a[5],a[10]....a[5n]比较,a[1],a[6],a[11]...a[1+5n]比较,直到a[4],......a[4+5n]比较
	第二趟：5/2间隔排序
	第三趟：5//2间隔排序
	第N趟: 1间隔排序

**第 0005 题:**  直接插入排序（Insert sort） 稳定
	第一趟: 创建一个数组,a[0]开始进入数组,但由于数组为空.a[0]不需要判断
	第n趟,a[n]进入数组,与数组中的数从后往前判断,如果a[n]小于a[x],则x和x之后的数在数组位置都要+1用于移除空位,让a[n]插入到a[x],如果a[n]大于a[max],则a[n] = a[max+1]
	最好情况:顺序 时间复杂度  T = O(n)	
	最坏情况:逆序 时间复杂度  T = O(n**2)

**第 0008 题:**  简单选择排序 (simplese lection sort)
	第一趟:循环数组,找到数组中最小的数,与a[0]交换
	第二趟:a[1]到a[max],找到其中最小的数,与a[1]交换
	第n趟: a[n-1]到a[max],找到其中最小的数,与a[n-1]交换
	无论如何 T = O(N**2)
